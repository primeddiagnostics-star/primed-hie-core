# Primed HIE Core

Primed HIE Core is an open-source Health Information Exchange (HIE) infrastructure designed to enable interoperable, scalable, and low-resource healthcare systems.

## Overview

The project provides foundational components for:

- Unique patient identity management
- Interoperable APIs for cross-facility data exchange
- Appointment and referral coordination
- Offline-first data capture and synchronization
- Secure and privacy-aware health data handling

## Purpose

This repository is part of an initiative to build digital public infrastructure for healthcare systems, particularly in low- and middle-income countries where fragmented systems limit continuity of care.

## Key Features

- Multi-tenant architecture supporting multiple hospitals
- Offline-first capabilities for low-connectivity environments
- Modular and lightweight deployment
- API-first design for easy integration
- Scalable infrastructure for national and regional health systems

## Use Cases

- National Health Information Exchanges (HIE)
- Maternal and child health tracking
- Cross-hospital referrals and continuity of care
- Public health data aggregation and planning

## Status

This project is actively being developed and deployed in real-world healthcare environments, with ongoing efforts to expand its open-source components.
